# Makka - Premium Maize Business Website

Frontend application built with Next.js, Tailwind CSS, and Framer Motion.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

```bash
npm run build
npm start
```

## Project Structure

```
src/
├── app/              # Next.js app directory and pages
├── components/       # Reusable React components
├── sections/         # Page sections (Hero, About, Products, etc.)
├── services/         # API integration services
├── context/          # React context for state management
├── hooks/            # Custom React hooks
├── utils/            # Utility functions
├── styles/           # Global styles
├── types/            # TypeScript types
└── assets/           # Images, icons, fonts
```

## Features

- Modern, responsive design
- Tailwind CSS styling
- Framer Motion animations
- JWT authentication
- Admin dashboard
- Product management
- Enquiry form system
- WhatsApp integration
- Contact management

See `.env.local` for configuration options.

## Technologies

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion

- **Icons**: React Icons
- **Forms**: React Hook Form
